# Isaac Clone - Игра в стиле The Binding of Isaac

Простая 2D игрушка в стиле The Binding of Isaac, написанная на Python с использованием библиотеки Pygame.

📓 База знаний проекта — Obsidian-vault прямо в этой папке (`.obsidian/`). См. [Changelog.md](Changelog.md), [Known_Issues.md](Known_Issues.md) и [Setting.md](Setting.md) (дизайн-документ сеттинга).

## Особенности

- **Игрок**: Движение WASD, стрельба стрелками (4 направления) или мышью (360°, зажатая ЛКМ)
- **Враги**: Разные типы врагов с уникальным поведением (базовые, стрелки, танки)
- **Предметы**: Собирайте предметы для улучшения характеристик
- **Подземелье**: Процедурно генерируемая сетка комнат (5×5) с дверями, плавными переходами, сокровищницей и боссом. Двери запираются, пока в комнате есть живые враги
- **Этажи**: После победы над боссом открывается люк — переход на новый этаж с сохранением характеристик игрока
- **Сундук-рулетка**: После каждого босса рядом с люком появляется сундук с одним из 10 уровней редкости приза (1 — частый и слабый, 10 — редкий джекпот). При касании — анимация рулетки, надпись с названием приза
- **UI**: Отображение здоровья, статистик, номера этажа, мини-карты подземелья и подсказок

## Установка и запуск

1. **Установите Python 3.7+**

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите игру:**
   ```bash
   python main.py
   ```

## Управление

- **WASD** - Движение игрока (стрелки НЕ двигают персонажа, только стреляют)
- **Стрелки** - Стрельба (4 направления, целится по последней зажатой стрелке)
- **ЛКМ (удержание)** - Стрельба в сторону курсора (360°, приоритет над стрелками пока зажата)
- **ESC** - Пауза/продолжить игру
- **R** - Рестарт (только на экране Game Over)

## Типы врагов

- **Красные (Basic)** - Простые враги, преследуют игрока
- **Синие (Shooter)** - Держат дистанцию и стреляют в игрока снарядами
- **Серые (Tank)** - Медленные, но прочные враги

## Предметы

- **Зелёные** - Восстанавливают здоровье
- **Красные** - Увеличивают урон
- **Синие** - Увеличивают скорость движения
- **Жёлтые** - Увеличивают скорострельность
- **Розовые** - Увеличивают максимальное здоровье
