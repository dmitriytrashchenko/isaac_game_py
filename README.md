# Isaac Clone - Игра в стиле The Binding of Isaac

Простая 2D игрушка в стиле The Binding of Isaac, написанная на Python с использованием библиотеки Pygame.

📓 База знаний проекта — Obsidian-vault прямо в этой папке (`.obsidian/`). См. [Changelog.md](Changelog.md), [Known_Issues.md](Known_Issues.md), [Setting.md](Setting.md) (дизайн-документ сеттинга), [Balance_Parameters.md](Balance_Parameters.md) (все проценты/множители игры), [Enemies.md](Enemies.md) (справочник типов врагов), [Session_2026-07-20_Visual_Overhaul.md](Session_2026-07-20_Visual_Overhaul.md) (визуальный апгрейд) и [Session_2026-07-21_Torch_Challenge_Stats.md](Session_2026-07-21_Torch_Challenge_Stats.md) (торч-челлендж, система статов).

## Особенности

- **Меню**: Главное меню → Настройки (полноэкранный режим, размер окна) / Выбор героя → игра. Управление клавиатурой и мышью, иконки героев — те же силуэты, что в игре
- **3 героя на выбор**: Лучник (дальний бой стрелами, база), Воин (ближний бой мечом, урон ×3, щит гасит слабые удары), Маг (файрболы — медленнее и реже, но урон ×2 и крупнее)
- **Игрок**: Движение WASD, атака стрелками (4 направления) или мышью (360°, зажатая ЛКМ) — тип атаки зависит от выбранного героя
- **Враги**: 5 типов с уникальным поведением и силуэтом — Скелет-воин, Гоблин-лучник, Каменный голем, Крыса-разведчик (быстрая и хрупкая), Громила (медленный, бьёт очень больно). Могут "воровать" брошенные предметы (25% шанс) и получают бафф вместо игрока
- **Предметы**: Силуэты вместо квадратов (флакон/клинок/сапог/молния/сердце), собирайте для улучшения характеристик
- **Вазы**: Разрушаемые препятствия (аналог горшков Isaac) — блокируют проход, разбиваются с одного попадания, шанс 40% оставить предмет. Разные палитры/формы/узоры
- **Валюта**: Глаза — выпадают только с врагов (20% шанс), не с ваз. Копятся на счётчике, магазина пока нет
- **Разрушаемые факелы**: Выдерживают 4 попадания, тускнеют по мере урона. Потушив ВСЕ факелы комнаты — шанс 15% на усиленную волну врагов (×2 или ×3 хп в зависимости от того, потушены ли факелы до или после зачистки обычных врагов), в боссовой комнате волна дополнительно призывает ещё одного босса. Награда — один сундук на всю волну; см. [Balance_Parameters.md](Balance_Parameters.md)
- **Подземелье**: Процедурно генерируемая сетка комнат (5×5) с дверями, плавными переходами, сокровищницей и боссом. Двери запираются, пока в комнате есть живые враги (в т.ч. повторно — во время усиленной волны). Факелы, тон палитры стен/пола и колонны-препятствия рандомизируются по комнатам
- **Этажи**: После победы над боссом открывается люк — переход на новый этаж с сохранением характеристик игрока
- **Сундук-рулетка**: После каждого босса рядом с люком появляется сундук с одним из 10 уровней редкости приза (1 — частый и слабый, 10 — редкий джекпот). Также редкий шанс (0.1%) выпасть с любого обычного врага, и гарантированный (тир 1-5) — награда за усиленную волну факелов. При касании — анимация рулетки, надпись с названием приза
- **Статы (Isaac-style)**: Видимые в HUD — Health/Speed/Damage/Tears/Range/Shot Speed/Luck; скрытые — Shot Height (снаряды пролетают препятствия при высоком значении), Knockback, Damage Multiplier
- **Атмосфера**: Процедурные факелы (мерцающее пламя + тёплое свечение), частицы (искры, вспышки попаданий), screen shake при уроне/победе над боссом
- **UI**: Отображение здоровья, полной панели статистик, номера этажа, счётчика валюты, мини-карты подземелья и подсказок

## Структура проекта

```
isaac_clone/
├── main.py          # Точка входа: главный цикл, переключение экранов, скейлинг рендера
├── menu.py          # Главное меню, настройки, выбор героя
├── viewport.py      # Масштаб/смещение логического кадра под реальное окно (для мыши)
├── game.py          # Класс игры и основная логика
├── player.py        # Класс игрока (в т.ч. механика 3 героев)
├── enemy.py         # Классы врагов
├── tear.py          # Снаряды (слёзы/файрболы) и вспышка удара мечом
├── room.py          # Класс комнат (текстуры, факелы, колонны, тон палитры)
├── item.py          # Класс предметов
├── chest.py         # Сундук-рулетка с призами после босса
├── vase.py          # Разрушаемые вазы-препятствия
├── currency.py      # Валюта (глаза)
├── particles.py     # Частицы (искры, вспышки попаданий)
├── ui.py            # Пользовательский интерфейс
├── constants.py     # Константы игры
├── requirements.txt # Зависимости
└── README.md        # Этот файл
```

## Установка и запуск

1. **Установите Python 3.7+**

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите игру:**
   ```bash
   python main.py
   ```

## Управление

**В меню:** стрелки/WASD + Enter, или мышь (наведение + клик)

**В игре:**
- **WASD** - Движение игрока (стрелки НЕ двигают персонажа, только атакуют)
- **Стрелки** - Атака (4 направления, целится по последней зажатой стрелке)
- **ЛКМ (удержание)** - Атака в сторону курсора (360°, приоритет над стрелками пока зажата)
- **ESC** - Пауза/продолжить игру
- **R** - Рестарт (только на экране Game Over)

## Типы врагов

- **Скелет-воин (Basic)** - Простые враги, преследуют игрока
- **Гоблин-лучник (Shooter)** - Держат дистанцию и стреляют в игрока снарядами
- **Каменный голем (Tank)** - Медленные, но прочные враги
- **Крыса-разведчик (Fast)** - Очень быстрая, но хрупкая (1 HP)
- **Громила (Brute)** - Медленный, но бьёт очень больно вплотную

## Предметы

- **Флакон (Health)** - Восстанавливает здоровье
- **Клинок (Damage Up)** - Увеличивает урон
- **Сапог (Speed Up)** - Увеличивает скорость движения
- **Молния (Tears Up)** - Увеличивает скорострельность
- **Сердце (Max Health)** - Увеличивает максимальное здоровье
- **Клевер (Luck Up)** - Увеличивает Luck (шансы всех дропов)
- **Мишень (Range Up)** - Увеличивает дальность полёта снарядов
