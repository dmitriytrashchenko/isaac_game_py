# Isaac Clone - Игра в стиле The Binding of Isaac

Простая 2D игрушка в стиле The Binding of Isaac, написанная на Python с использованием библиотеки Pygame.

## Особенности

- **Игрок**: Движение с помощью WASD, стрельба стрелками
- **Враги**: Разные типы врагов с уникальным поведением (базовые, стрелки, танки)
- **Предметы**: Собирайте предметы для улучшения характеристик
- **Комнаты**: Процедурно генерируемые комнаты с врагами
- **UI**: Отображение здоровья, статистик и подсказок

## Структура проекта

```
isaac_clone/
├── main.py          # Основной файл запуска игры
├── game.py          # Класс игры и основная логика
├── player.py        # Класс игрока
├── enemy.py         # Классы врагов
├── tear.py          # Класс снарядов (слёз)
├── room.py          # Класс комнат
├── item.py          # Класс предметов
├── ui.py            # Пользовательский интерфейс
├── constants.py     # Константы игры
├── requirements.txt # Зависимости
└── README.md        # Этот файл
```

## Установка и запуск

1. **Установите Python 3.7+**

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите игру:**
   ```bash
   python main.py
   ```

## Управление

- **WASD** - Движение игрока
- **Стрелки** - Стрельба в соответствующем направлении
- **ESC** - Пауза/продолжить игру

## Типы врагов

- **Красные (Basic)** - Простые враги, преследуют игрока
- **Синие (Shooter)** - Стреляют в игрока, держат дистанцию
- **Серые (Tank)** - Медленные, но прочные враги

## Предметы

- **Зелёные** - Восстанавливают здоровье
- **Красные** - Увеличивают урон
- **Синие** - Увеличивают скорость движения
- **Жёлтые** - Увеличивают скорострельность
- **Розовые** - Увеличивают максимальное здоровье
