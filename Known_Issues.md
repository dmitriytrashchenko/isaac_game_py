# Known Issues

Актуальный список недоделок и известных проблем. См. также [Changelog.md](Changelog.md) для истории фиксов.

## Недоделки

- **R — рестарт по тапу, а не по удержанию 3 сек** — в оригинале Isaac рестарт требует удержания клавиши ~3 секунды. Сейчас у нас срабатывает по одиночному нажатию. Отложено как некритичное.
- **Нет звука** — `pygame.mixer` не используется вообще.
- **Нет анимации спрайтов** — все объекты — залитые прямоугольники, меняющие цвет (не считается анимацией в привычном смысле).

## Мелкая неэффективность (не критично)

- `Tear.update()` создаёt новый `pygame.Surface` каждый кадр для альфа-затухания вместо переиспользования поверхности или `set_alpha()`.

## Design-декларации (не баг, но стоит знать)

- Диагональные слёзы от импульса движения реализованы с весом `momentum_weight = 0.35` — подобрано на глаз, не сверено с точным поведением оригинала (там нет открытого исходника для сверки).
